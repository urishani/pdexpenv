Your goal is to correctly complete as many tasks as you can in the time you have. Each task requires you perform an action on a scope of cells in the table.
 
There are 3 types actions:
        Addition - Adding a number to the current value of cells.
        Multiply - Multiplying the current value of cells.
        Remainder - Placing a value in cells when the current value of cells is divisable by         a number without a remainder.
And 3 types of scopes:
        Single Cell - Apply the action to a specific cell in the table.
        Part of a Row - Apply the action to a series of cells on a specific row.
        An area - Apply the action to a series of cells on a series of rows (a rectangular         area of the table).
 
The combination of actions and scopes gives 9 types of tasks.

Examples:
        Addition action on the scope of a row:
        Add 9 to the value of each cell located on row 12 between column 3 and column 12.
        You have to add 9 to the current value of the the part of row 12 that is between         columns 3 and 12.
 
        Division action on the scope of an area:
        Put the value 999 in each cell that is located between row 7 and row 12 between         column 7 and column 13 if it can be divided by 3 without leaving a remainder.
        You have to check each cell in the designated area, if it can be divided by 3 without         leaving a remainder, put the value 999 in the cell, if not, leave it unchanged.
