The system uses wizards to allow you to manipulate the data in the table. Once you enter the parameters click "Execute" to change the data in the table. The "Show code" button will show you the code the wizard generates and runs to make the changes to the data in the table (You must click "hide code" in order to be able to click "Execute").

The 9 wizards do either of three actions:
    *   Addition: adds a number to the current value of cells.
    *   Multiplcation: Multiplies the current value of cells by a number.
    *   Remainder: Places a selected value in cells that can be divided by a number without         leaving a remainder
These three actions can be applied to cells in the table in either of three ways:
    *   A single cell: You specify the cell to change using the row and column numbers it         is located on.
    *   By Row: Will change cells on a specified row or part of a row, using the row         number and a range of column numbers they are located on.
    *   An Area: Will change cells in a rectangular area of the Table, using a range of row         numbers and a range of column numbers they are located on.

If the range of rows and/or columns have an error, you will be notified in the messages window.

Possible errors are the result of the following:
    *    If you do not use a positive whole number.
    *    Entering a row or column number that are beyond the range of the table (i.e.          entering row number 17 when there are only 15 rows).
    *    The ending row or column number are lower than the starting row or column          number (i.e. from row 10 to 8).
