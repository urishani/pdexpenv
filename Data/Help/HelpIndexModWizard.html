This Help system has information that you can use to complete the tasks you are given.
You can start it at any point by using the Green help button located above the table.

There are two main sections:
    *   About the system: introduces the system, has info on what all the different windows       do and what wizard tools are available.

    *   About the tasks: describes the tasks and special buttons that relate to them.

Use the links at the top of the page to move between help topics.

In some cases, there will be a list of sub-topics on the left side of the page.

You can close the help system using the "QuitHelp" button in order to return to the system and continue your work.
