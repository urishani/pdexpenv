Help Topics:

This is the help system for this programming environment.

To the left is a list of topics that this help file can help you with.

  • About the system or: "What everything is": This has information about the system, what all the different windows are and what they are for.
  • About the tasks or: "What you need to do": This section gives you information about the what you are supposed to do, and how to use the system to complete the tasks you have to do.
  • About writing code or: "How to write the code": This section gives you information on the different commands you can use in the code, how to use them and the rules on how to put it all together (syntax).
