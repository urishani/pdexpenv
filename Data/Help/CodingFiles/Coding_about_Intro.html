This system has all the tools and information you need in order to complete the tasks.

All the tasks involve changing data in a table of numbers.

The different areas of the screen have different purposes - they are described in the windows section.

You will use the system to know what the task you need to complete is.

You need to then write a short series of commands in a coding editor.

After you run the program the system will tell you if you completed the task correctly or not.

The help system can be used at any point to get information that you need to complete the tasks.
