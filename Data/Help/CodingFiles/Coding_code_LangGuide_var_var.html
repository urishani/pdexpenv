Variables:
Are a basic element of the language. They are containers that hold data (e.g. numbers).
 
You create a variable by giving it a name and placing a value in it.
 
You can use any name for a variable that is not RESERVED as a word used by the language.

There are special variables that are part of the language, such as the table variable.
A variable can store only one value at a time. the variable's value remains the same untill it is replaced with a new one.
 
A number stored in a variable can be manipulated by arithmetic expressions (just like a regular number is).
