The table[ , ] command lets you use a special variable called the table variable. This is a variable that holds the value inside a specific cell in the table of numbers. Any time you want to use the value in a cell in any type of statement (assignment statement, if statement, for statement), or expression (logical expression or arithmetic expression), you use the table[ , ] command.

The way to use it is:
table[row, column]

you can also use a variable inside the [] instead of numbers like: table[a, b]
