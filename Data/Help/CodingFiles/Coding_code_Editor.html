This code editor is made up of the Writing area, three buttons: Clear, Compile and Execute and the Messages window - that gives messages about the code and any errors that stop the code from running.
 
In order to use the code editor you must write the code in the code area, compile it with the Compile button and if there are no errors, you can run it by clicking the Execute button.
  
Each of the following sections gives detailed information about the different parts of the code editor system and how to use them to get the code to run and change the values of the cells in the table in order to complete the tasks.
 
 
 
 
 
 
You can navigate the pages via the links in the left side panel.
 
Alternatively, you can use the "Previous" and "Next" links at the top to navigate the pages in their order.
