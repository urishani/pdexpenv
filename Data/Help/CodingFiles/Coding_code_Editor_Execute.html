* Execute button
 
* Messages
 
 

 

The Execute button is located below the Writing area, and next to the Clear and Compile buttons.
 
This button will appear as a grey button (meaning that it cannot be clicked) until the text in the Writing area is compiled and the system finds no errors in the code. After this the Execute button will turn to an enabled status (meaning that it can be clicked).
  
If you click the Execute button, the code will be run and any changes that the code makes to the table will be made on the table. If there are any runtime errors, they will be displayed in the Messages window.
 
Once the Execute button is clicked it will become disabled again. You will need to compile the text again. If you make any changes to the text in the Writing area after you compile, you need to compile the text again before you can execute it again.
