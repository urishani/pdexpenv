The elseif command is part of the if statement. it is used only if you want to do one thing when the condition is true, and a different thing when the condition is false and a different condition is true.
the way to use it is:

  if condition do
     statements ;
  elseif condition do
     statements ;
  endif;

Comments:
condition --> is a logical expression like a>b or another condition like a<c
statements --> is a statement of what to do if the condition is true and another statement to do if the second condition is true (you can also use a list of statements instead of just one).

Example:
   if a>b do
    table[1,2]:=3 ;
  elseif a<c do
    table[1,2]:=4 ;
  endif ;
