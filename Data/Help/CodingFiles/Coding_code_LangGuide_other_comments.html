Comments can be added to the code you write. Adding comments to the code may help you or others to understand what the code does. Anything you write in a comment will not be compiled by the system - and will not affect the code in any way. You can write a comment in the same line as code, or in a new line.

The syntax is:
  # comment text ;
  
Where:
   # begins the comment.
   comment text is the text you want to write as a         comment.
   ; ends the comment.
 
Example:
  
  Column := 4 ;
  # on the row above is the variable for ;
  # columns in a table ;

 A comment on the same line as code:
  Row := 3; # this is a variable for rows in ;   # a table ;
