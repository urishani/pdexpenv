The for statement repeats (loops) a set of statements depending on the value of a variable that has 1 added to it each time the statements are repeated.

The syntax is:
  for variable := parameter to parameter loop
     statement ;
  next variable ;
Where:
   for - starts the iterative statement.
   Variable - is the name of the variable.
   parameter - are the start and end values of the variable         (must be positive whole numbers).
   loop - after this you list the statements to do
   statement   - is the list of statements (each ending in ;).
   next variable - ends the loop. variable needs to be        the same as the variable at the beginning of the loop.

Example:
   for Row:=1 to 3 loop
    table[Row, 1] := 999;
  next Row ;

Nested loops example:
  for Row:=1 to 3 loop
    for Column:=4 to 8 loop
      table[Row, Column] := 999 ;
    next Column ;
  next Row ;
