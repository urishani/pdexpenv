*Runtime Errors
 

 

This section lists the different error messages you can get from the system after executing the code you created.
 
In some cases, the code will be compiled correctly and you will be able to Execute it, but you will recieve an error. Thsi type of error is a "Run Time Error", it is called that because it happens when you RUN the program.
 
These errors will cause the program to stop and an error message will appear. But, the task will appear as completed incorrectly and you will be moved to the next task.
 
The errors you can get are:
*
Exception Var __ is not defined - Using a variable in an expression without defining it.
Example: writing Table[a, 1]:=999; without defining the variable 'a'
 
*
Indexes are out of bounds - Referencing a cell in the table that is not inside the table will give an error.
Example: writing Table[40, 1] where the table has only 35 rows.
 
*
Arithmetic Exception - if you divide by zero, you will get this error.
