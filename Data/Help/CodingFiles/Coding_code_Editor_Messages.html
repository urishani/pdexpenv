* Messages
 
 

 

The messages that the system prints are displayed in the Messages window, and messages about completing the task or making an error with the task are displayed in a pop-up window.
 
The messages in the Message window are printed either in:
* Green - if they are ok
or:
* red - if they are error messages.
 
Additional information on the error messages you can get appears on the Language guide
