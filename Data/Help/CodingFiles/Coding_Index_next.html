The next command is part of the for statement.
the way to use it is:
  for variable := parameter to parameter loop
     statement ;
  next variable ;
another way to use it is with two loops (called "nested loops"):
  for variable1 := parameter to parameter loop
    for variable2 := parameter to parameter loop
         statement ;
      next variable2 ;
  next variable1 ;

Comments:
variable --> is the variable that will have its value changed each time the loop repeats (each time it repeats, the value will have 1 added to it).
parameter --> is first: the number from which to start and second: the number on which to end
statement --> is a statement of what to doeach time the loop repeats (you can also use a list of statements).

Examples (first regular loop and then a pair of nested loops):
   for a:= 1 to 5 loop
    table[1,a]:=10 ;
  next a ; 

   for a:= 1 to 5 loop
     for b:= 1 to 5 loop
      table[b,a]:=10 ;
    next b ;
  next a ;
