 *  Table Window
 
The Window located at the bottom left of the screen is the Table Window. A table of randomly generated numbers is located in this window.
 
Changes made to the content of a cell in the table are displayed and the cell will change its background color to yellow.
 
You can use the "Reset Table" button located a the top of the window to undo any changes made to the table (see the page about reset table for more information).
