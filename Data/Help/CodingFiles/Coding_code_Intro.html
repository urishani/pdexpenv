This system uses a very simple programming language that changes data in the table of numbers in the Table window. The code editor is a simple system: you can read about it here.
 
* The specific commands and rules about how to write code are detailed in the Language guide.
  
A program: a list of statements that tell the system what to do.
 
Compiling: For a program to run it needs to be compiled - this process translates the statements into instructions that make the actual calculations and changes to the cells in the table.
 
Compiliation errors: If any of the statements have errors that make it impossible to compile the program, a Compilation error results and the program needs to be corrected.
 
Executing a program: After the program is compiled it can be Executed. The system will follow the instructions of the compiled program and make changes to the table accordingly.
 
Run-Time Errors: If the program has errors that make it impossible to Execute (like trying to change a cell in row #16 when the table only has 15 rows) a Run-Time error results and the program needs to be corrected.
