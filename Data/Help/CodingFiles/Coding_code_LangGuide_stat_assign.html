An assignment statement puts a value in a variable. The value will be used when you use the variable later.

First you state the name of a variable you want to assign a value to, and then state the value you want to assign to it.

The syntax is:
  Variable := Value ;
Where:
   Variable is the name of the variable.
   Value is the value that is put in the variable.
   :=    is what assigns the value to the variable
   ;    is the end of the statement.

Example:
In order to assign the value 3 to the variable Row, use:
  Row:=3;

You can also use the variable in an arithmetic expression like this (Row will hold the value 4):
  Row:=3;
  Row:=Row+1;

Note: If you try to use a reserved word as a variable in an assignment statement, you will recieve an error.
