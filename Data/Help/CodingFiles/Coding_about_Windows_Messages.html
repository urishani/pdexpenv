The Window located at the top right of the screen is the Messages Window. This is where the system displays any messages or errors after compiling or executing code.
 
You can scroll back to previous messages with the scroll bar or clear the messages from the window with the "Clear Messages" button located at the top of the window.
 
Messages are printed in Green. Errors are printed in Red. You can read more about Messages and Error Messages in the Code editor section.
 
A list of errors and their types are located in the Language guide section.
