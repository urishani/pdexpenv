The table is a matrix of cells, organized in Rows and Columns. Each cell is a variable that can hold a number. Anything you can do to a variable, you can also do to a cell.
Each cell is named by the row and column in the table that it is located on.

The syntax is:
table[Row, Column]
Where:
  Row  is the row number of the cell.
  Column  is the column number of the cell.

Example:
Refferencing Cell on row 3 and column 2:
table[3, 2]
Note: Row and Column are positive numbers, that cannot exceed the respective table width and height.
The table width and height are stored in the special read-only variables:
table.m or table.x - the number of columns (table width)
table.n or table.y - the number of rows (table height)
