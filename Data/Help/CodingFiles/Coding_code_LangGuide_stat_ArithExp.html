An Arithmetic expression performs arithmetic operations on variables and numbers. An expression can be used in statements, such as assignment statements.

The operators are:
  +  -  *  /  REM
Where:
   +       is addition.
   -        is subtraction.
   *       is multiplication.
   /        is division.
   REM   is modulus (the result is the remainder of the                          division of two numbers).

Examples:
An expression in an assignment statement (Row will be 6):
  Row:= 3 + 3 ;
Order of operations (Column will be 17):
  Column:= 3 * 4 + 5 ;
Using Modulus (x will be 0):
  x := 4 REM 2 ;
